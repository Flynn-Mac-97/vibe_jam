# Theme Convergence · 15-Minute Protocol

廖琦 runs this, timer visible to all. If minute 15 ends and no decision is made, 廖琦 forces one. No re-openings.

## Inputs

- The announced theme (one or two words).
- All three playbooks in `04_design/PLAYBOOK_A/B/C.en.md` (and their `.zh.md` twins, in 廖琦's view only).
- The existing night-sky scene and assets.

## The 15 minutes

### Minute 0–3 · Theme reading (3 min)

Silent write (everyone). Each person writes 3 one-line interpretations of the theme. No commentary, no discussion.

Template:
- Interpretation 1: …
- Interpretation 2: …
- Interpretation 3: …

At minute 3, 廖琦 collects. Do not read them all aloud. Pick the 5 most distinct (discard duplicates).

### Minute 3–8 · Map to playbooks (5 min)

On a shared view (whiteboard / shared doc), 廖琦 places the 5 interpretations on a 3-column table:

| Interpretation | Best fits Playbook | Why (1 line) |
|---|---|---|
| … | A / B / C | … |

Constraint: each interpretation goes to exactly one playbook. If an interpretation fits two, pick the more reliable one (A > C > B for reliability).

At minute 8, count: which playbook gets the most interpretations? That's the winner.

Tiebreaker rule: if two playbooks tie, A wins. If A and B tie, A wins. If A and C tie, A wins. If B and C tie, B wins. (A is the most reliable path. Default toward reliability.)

### Minute 8–13 · Commit 3 must-do / 3 won't-do (5 min)

Open the chosen playbook. Its base must-do / won't-do list is the starting point.

Team rewrites both lists **with theme-specific additions**:

**Must do** (exactly 3):
1. …
2. …
3. …

**Won't do** (exactly 3):
1. …
2. …
3. …

These 3 + 3 are the contract for the rest of the day. No one adds to either list without a scope-swap: remove one item to add another.

### Minute 13–15 · One-sentence definition + kick off (2 min)

廖琦 writes one sentence on the board:

```
In [setting], the player [core verb] [target] to [goal], while [pressure].
```

Example (if theme is "Loop" and playbook is A):
```
In the looping night sky, the player collects falling stars to beat their best score, while meteors from earlier loops fall faster each cycle.
```

Read it aloud once. Everyone confirms or raises one objection. 廖琦 decides.

At minute 15: Flynn starts importing scripts. David starts scaffolding the gameplay files needed for the playbook. mengping identifies which existing asset will be the hero. trixie starts BGM for `BgmKey.Main`.

## Rules

- Nobody starts coding or opening Unity until minute 15. Convergence first.
- No phones during this 15 minutes.
- If the theme seems impossible to map, pick Playbook A and find any metaphor. Shipping beats clever.
- If someone says "but what if we did something totally different" after minute 15, the answer is no.

## Convergence rehearsal (Saturday night)

廖琦 picks a fake theme at random ("symbiosis", "fragment", "loop", "light", "small things matter"). Team runs the full 15 minutes. If the team can't commit at minute 15 during rehearsal, redo the rehearsal. The day-of has no margin.
