# Using GitHub Desktop to Edit, Save, Push, and Pull

## Getting Started

1. **Clone the Repository**
   - Open GitHub Desktop
   - Click **File > Clone Repository**
   - Select `Flynn-Mac-97/vibe_jam` and choose a local path
   - Click **Clone**

## Editing Files

1. Click **Open in Visual Studio Code** (or your preferred editor) from GitHub Desktop
2. Make your changes to any files in the project
3. Save your files (`Ctrl + S`)

## Committing and Pushing Changes

1. Open **GitHub Desktop** — you'll see your changed files listed on the left
2. Review the changes shown in the diff view on the right
3. Check the boxes next to the files you want to include
4. At the bottom left, enter a **Summary** (short description of what you changed)
5. Optionally add a **Description** for more detail
6. Click **Commit to main**
7. Click **Push origin** at the top to upload your changes to GitHub

## Pulling Changes

Before you start working, always pull the latest changes:

1. Open **GitHub Desktop**
2. Click **Fetch origin** at the top to check for updates
3. If changes are available, click **Pull origin** to download them

> **Tip:** Always **pull before you push** to avoid merge conflicts.

## Quick Workflow Summary

1. **Pull** — Get the latest changes
2. **Edit** — Make your changes in your editor
3. **Save** — Save your files
4. **Commit** — Write a summary and commit in GitHub Desktop
5. **Push** — Push your commit to GitHub
